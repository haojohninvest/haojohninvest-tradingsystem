# Django Admin 帳號資訊

## 登入網址
http://localhost:8000/admin/

## 帳號密碼

| 用戶名 | 密碼 | 權限 |
|--------|------|------|
| Johnny | 0002 | 超級用戶（完全控制） |
| William | 0001 | 一般用戶（可編輯 StockSector） |

## 使用方式

### 1. 登入 Admin 後台
1. 打開瀏覽器訪問：http://localhost:8000/admin/
2. 輸入用戶名和密碼
3. 登入成功

### 2. 編輯股票分類
1. 點選「股票分類對應」
2. 搜尋股票（輸入代號或名稱，例如 `2330` 或 `台積電`）
3. 點選該股票
4. 在「所屬分類」下拉選單選擇新分類
5. 按「儲存」

### 3. 批量修改
1. 勾選多支股票（左上角可全選）
2. 頁面下方動作選擇「變更為所屬分類」
3. 按「確定」
4. 選擇新分類並確認

### 4. 查看修改紀錄
1. 在 Admin 首頁點選「Log entries」（日誌記錄）
2. 可看到所有修改歷史（誰、什麼時候、改了什麼）

## 功能說明

✅ 可搜尋股票（代號/名稱）
✅ 可依分類篩選
✅ 可修改單一股票分類
✅ 可批量修改多支股票
✅ 自動記錄誰改了什麼
✅ 分頁顯示（每頁 100 筆）

## 注意事項

- 請不要將密碼告訴其他人
- 修改紀錄會永久保存
- 如需修改密碼，請在 Admin 後台「使用者」中設定
