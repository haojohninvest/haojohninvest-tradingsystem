## 一、專案背景

### 現有系統（Observation_2024.py）
- 從 TWSE / OTC 爬取台灣上市櫃股票每日 OHLCV 資料
- 計算技術指標：EMA20/60/120、SMA20/60/120、當日漲跌、五日漲跌
- 計算 Market Breadth（市值前 200 大，收盤 > 20 日均線的比例）
- 計算 Sector Divergence（產業族群市值加權 EMA20 偏離度）
- matplotlib 靜態圖表輸出
- Jupyter Notebook 手動執行，CSV 檔案儲存

### 現有問題
1. 手動執行 - 每天要自己開 Notebook 跑
2. 單機運行 - 資料、程式都綁在個人電腦
3. 不可分享 - 別人看不到分析結果
4. 程式碼結構 - 大量重複邏輯（TWSE/OTC 幾乎一樣的 code 複製貼上）
5. CSV 儲存 - 查詢慢、不可靠、檔案越來越大

---

## 二、目標

將現有 Jupyter Notebook 交易觀測系統，升級為：
- 自動排程執行的 Web 應用程式
- 朋友可以登入、互動、留言
- 可在網頁上調整參數（日期區間、公司分類等）
- 特定條件觸發時透過 LINE 通知

## 三、技術棧

| 項目 | 技術選擇 | 說明 |
|------|---------|------|
| 後端 | Django 5.x + Python 3.11+ | 全功能框架，內建用戶系統、ORM、管理後台 |
| 前端 | Django Templates + HTMX + Tailwind CSS | 幾乎不用寫 JS，Python 工程師友好 |
| 圖表 | Plotly | 互動式圖表，可縮放、hover 看數值 |
| 資料庫 | PostgreSQL (production) / SQLite (development) | 取代 CSV，查詢快、可靠 |
| 排程 | django-apscheduler | 每日自動爬取 + 計算 |
| 通知 | LINE Notify API | 條件觸發推送 |
| 部署 | Railway / Render (~$5-7/月) | 雲端部署 |
| 版控 | Git + GitHub | 版本管理 |

### 為什麼選 Django？
- **內建用戶系統**：登入/註冊/權限，幾行設定就搞定
- **內建管理後台**：可以直接在後台看/改資料庫資料
- **內建 ORM**：用 Python 語法操作資料庫，不用學 SQL
- **模板系統**：寫 HTML 就能出頁面，搭配 HTMX 幾乎不用寫 JS
- **社群大**：教學資源多、中文資源豐富

### 為什麼不選其他？
- **Streamlit**：無法做用戶系統和留言互動功能
- **FastAPI**：只做 API，前端要另外用 React/Vue 寫，工作量翻倍

## 五、資料庫設計（取代 CSV）

```
Models:

Stock（股票基本資料）
├── code          : CharField (4碼，如 "2330")
├── name          : CharField (如 "台積電")
├── market        : CharField ("twse" / "otc")
├── market_cap    : BigIntegerField (市值)

DailyPrice（每日 OHLCV）
├── stock         : ForeignKey → Stock
├── date          : DateField
├── open          : DecimalField
├── high          : DecimalField
├── low           : DecimalField
├── close         : DecimalField
├── volume        : BigIntegerField (成交金額)

Indicator（技術指標，每日每股一筆）
├── stock         : ForeignKey → Stock
├── date          : DateField
├── ema20         : DecimalField
├── ema60         : DecimalField
├── ema120        : DecimalField
├── sma20         : DecimalField
├── sma60         : DecimalField
├── sma120        : DecimalField
├── daily_return  : DecimalField (當日漲跌)
├── five_day_return : DecimalField (五日漲跌)
├── ema20_dev     : DecimalField (EMA20偏離率)
├── ema60_dev     : DecimalField (EMA60偏離率)
├── ema120_dev    : DecimalField (EMA120偏離率)

Sector（產業分類）
├── name          : CharField (主分類名稱)
├── category_type : CharField ("主分類" / "子分類")

StockSector（股票-分類對應，可在網頁上編輯）
├── stock         : ForeignKey → Stock
├── sector        : ForeignKey → Sector
├── modified_by   : ForeignKey → User (誰修改的)
├── modified_at   : DateTimeField (修改時間)

Message（留言）
├── user          : ForeignKey → User
├── content       : TextField
├── created_at    : DateTimeField
├── page          : CharField (在哪個頁面留言)

StockWatch（個股註記 / 追蹤）
├── user          : ForeignKey → User
├── stock         : ForeignKey → Stock
├── note          : TextField (備註)
├── created_at    : DateTimeField
```
